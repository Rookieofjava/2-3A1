介紹
---
整理餐廳口袋名單的好幫手，可瀏覽、查詢及連結地圖導航
### 功能
- 顯示所有餐廳清單
- 呈現餐廳詳細資料
- 餐廳地址連結google map
- 以關鍵字或是類別搜尋餐廳




開始使用
---
1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：
```javascript=
    npm install
```
4. 安裝完畢後，繼續輸入:
```javascript=
    npm run start
```
5. 若看到此行訊息則代表順利運行，打開瀏覽器進入到以下網址
```javascript=
    Listening on http://localhost:3000
```
6. 若欲暫停使用
```javascript=
    ctrl + c
```
